# Pharmacy-Management-System
Full Stack Pharmacy Management System using React, Spring Boot, and MySQL with Admin Dashboard, Product Management, Cart, and Order Processing.


Default Admin Credentials
Username: admin
Password: admin

## 🚀 Features

### 👤 User Features
- User Registration & Login
- View all products
- Search products
- Add to cart
- Update quantity in cart
- Place orders
- View order history

---

### 🧑‍💼 Admin Features
- Admin Login
- Dashboard panel
- Add new products
- Update product details
- Delete products
- View all users
- View all orders

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Material UI (MUI)
- Axios

### Backend
- Spring Boot
- Spring Data JPA
- REST APIs

### Database
- MySQL

- Git & GitHub
- Postman

## ⚙️ Installation & Setup

### 🔹 Backend Setup

cd P_M_S
mvn spring-boot:run

👉 Runs on:

http://localhost:8080


Frontend Setup
cd pmsfront
npm install
npm run dev

👉 Runs on:

http://localhost:5173
